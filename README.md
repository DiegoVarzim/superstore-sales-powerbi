# Análise de Vendas - Superstore Sales (Power BI)

Projeto de portfólio de **Analista de Dados Júnior**.

## Objetivo do Projeto
Analisar o desempenho de vendas de uma rede de varejo americana entre 2014 e 2017, identificando padrões, oportunidades e insights de negócio.

## Principais Insights
- Tecnologia é a categoria mais rentável
- Segmento Consumer representa mais de 50% das vendas
- Crescimento consistente ao longo dos anos
- Forte concentração de vendas nos estados do leste dos EUA

## Tecnologias Utilizadas
- **Power BI Desktop**
- DAX (medidas e cálculos)
- Power Query (limpeza e transformação)
- Modelagem de dados (tabela de calendário e relacionamentos)
